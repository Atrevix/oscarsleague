Qui sommes-nous ?

Oscar's League est une équipe basée sur le jeu vidéo Fortnite et a été créée par Kaizer et Atrevix le 04/08/2024. Le but de l'équipe est de s'organiser pour rejoindre des cartes en mode créatif pour remplir celle-ci du tag « Rugissement d'Oscar ». Aucune confrontation, aucune action violente : nous sommes juste là pour le fun ! Les organisations se présentent en appel vocal sur notre serveur Discord avec une carte choisie à l'avance et démarrent dans un groupe créé sur Fortnite avec toutes les personnes présentes dans l'appel. Ensuite, l'attaque commence : tous les membres commenceront donc à remplir la carte des tags « rugissement d'Oscar ». Une fois que cela est fait, nous enregistrons en vidéo un extrait de la carte envahie par ces tags et nous la partageons en ligne sur nos réseaux sociaux (ci-dessous) pour partager à des particuliers notre divertissement !

Comment nous rejoindre ?

Pour nous rejoindre, vous devez d'abord rejoindre notre serveur Discord. Rendez-vous en premier sur le canal #exigences pour savoir si vous êtes éligible, puis si vous respectez les exigences, vous pouvez essayer de vous inscrire dans #inscriptions ! Si vous n'êtes pas éligible, pas de soucis : vous serez averti des événements quand les membres auront fini de remplir la carte de tags
